v1.4 Blueprints Tutorials

This page now loads tutorial bodies dynamically from tutorial_v1_4_sections.

Setup Ready

Add new HTML files inside tutorial_v1_4_sections, then register them in tutorial_v1_4_sections/manifest.json. The sidebar and section loading will update from that manifest.

Accepted Formats

You can save each tutorial file in either of these formats:

  • full section markup, for example <section class="tutorial-section">...</section>
  • raw HTML body content only, because the page will wrap it automatically if needed

Quick Example

manifest.json 
{
  "id": "installation",
  "label": "Installation",
  "file": "02_Installation.html"
}
